# Retell Agent Setup Instructions

## Manual Import Steps (Free Tier)

Since Retell API requires paid tier for programmatic access,
follow these manual steps to import the agent spec:

### Step 1
Go to retell.ai and log in

### Step 2
Click "Create Agent"

### Step 3
Copy the system_prompt from:
outputs/accounts/ben_electric/v1/agent_spec.json

### Step 4
Paste it into the Retell agent prompt field

### Step 5
Set the following:
- Agent Name: Clara - Ben's Electric Solutions
- Voice: Professional female voice
- Language: English

### Step 6
Save the agent

### Step 7
Repeat with v2 agent spec after onboarding update

## API Keys
- Retell API keys available in Settings > API Keys
- Free tier does not support programmatic agent creation
